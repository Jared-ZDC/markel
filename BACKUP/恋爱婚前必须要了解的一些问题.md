# 问题一： 父母的问题
* 婚后，我们能不能看重并尊敬对方的父母？
* 我们有没考虑到父母可能会干涉我们的关系？
* 我们父母的态度，有没有达到我们的预期？
* 会不会给足够的祝福？如没有，我们如何面对？
* 如果你的父母对你的爱人不敬，你该怎么处理？

# 问题二： 房子的问题
* 所以婚前要讨论好，是不是结婚前一定要买房（北上广深情侣常见问题）？
* 如果不买房先结婚，等攒好钱再买可不可以？
* 买房能接受多大平方的？（关乎未来谁来住）
* 位置能接受在哪里？（关乎你们的通勤时间）
* 装修风格谁来定？装修资金有多少？能否养宠物？
* 怎么拿首付？如果是双方共同还贷，房产证写谁的名字？

# 问题三： 孩子的问题
* 我们要不要孩子？要的话，想在什么时候？
* 未来孩子出生后，怎么带大？
* 找谁的父母帮忙？还是请月嫂？
* 双方教育孩子的思路有差异，怎么办？

# 问题三：婚后异性交往的底线在哪里
* 双方异性交往的空间有多大？
* 和异性同事或朋友能不能单独吃饭和聊天？
* 除了必须要参加的正式工作场合，其他聚会要不要带另一半？
* 不带另一半的情况下，出去喝酒吃饭，晚上几点前要回家？
* 双方的手机密码能不能知道？可不可以随便看？

# 问题四： 婚姻里需要约法三章的事
* 我们永远不会因为婚姻放弃的东西是什么？
* 我们清晰地了解对方的精神需求及信仰吗？
* 我们在婚姻里，绝对不能允许的事情是什么？
* 我们真的能倾听对方诉说，并公平对待对方的想法和抱怨吗？
* 我们是不是充满信心，即使面对任何挑战，也能让婚姻一直往前走？
* 如果有一方因自身原因提离婚（不爱了/出轨），那另一方的权益如何保障？
